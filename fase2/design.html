¿que es flex box?

La pagina somos pnt nos dice que el flex box es Principalmente es un modulo de diseño CSS3 que se diseño para facilitar la forma en que se hace responsive, evitando el uso de float y escribiendo menos codigo y facilitando el posicionamiento de los elementos, incluso sin saber el tamaño de estos. Basicamente es poder ajustar el ancho lo alto de nuestros elementos de una manera simple con el espacio que se dispone. (somospnt.com/blog2020)

Las listas y enlaces html

segun el portal web MANUAL WEB Las listas html nos permiten introducir conjuntos de elemento en orden de listas dentro de una pagina y normalmente estan ordenados o separados por un guion o un numero. existen varios tipos de listas html (listas ordenadas, listas desordenadas, listas de definiciones y listas anidadas) las listas ordenas html son las que nos muestran todos los elementos de la lista en orden, van en numero o por letras las listas html ordenadas se presentan mediante el elemento ol. Las listas desordenadas son las que se muestran sin ningun tipo de orden y van acompañadas por un punto un guion en su defecto. las listas por definiciones Las listas en HTML de definiciones en HTML nos sirven para montar listas en las que tenemos la estructura valor y definición. Suelen ser listas para definir términos, como si fuese un diccionario, si bien pueden ser cualquier par valor-definición. Cuando estemos manejando listas podemos anidar unas en otras independientemente del tipo de lista que estemos anidando.
